## [v1.0.0.0] - 2026-05-02

### 🎉 首个正式版本发布

**文件管理**
- 支持 .dat 格式文件导入（简单格式和详细格式）
- 支持创建空白文件并手动添加点位
- 文件回收站功能，支持误删恢复
- 文件导出功能，可自定义导出内容
- 投影参数配置（坐标系统、投影方式、中央经线）

**地图展示**
- 多种底图支持：OpenStreetMap、天地图（矢量/影像/地形）
- 天地图 Token 配置和使用说明
- 地图模式 / 网格模式切换
- 智能点位聚合，提升大数据量性能
- 点位标签悬浮显示（可选）

**点位管理**
- 点位类型：碎部点（蓝色）/ 控制点（红色）
- 点位操作：重命名、删除、类型切换
- 手动添加点位（基于当前位置或自定义坐标）
- 点位搜索和快速定位
- 点位筛选（全部/碎部点/控制点/手动点）
- 点位批量管理和编辑

**测量工具**
- 距离测量（空间距离和平面距离）
- 高差计算

**异常检测**
- 精度异常检测（HRMS/VRMS 超标）
- 孤立点检测
- 统计分析（总点数、异常数、平均精度）
- 异常分类展示和导出

**坐标系统**
- 支持多坐标系：CGCS2000、Beijing54、Xian80、WGS84
- 高斯-克吕格投影（3°带 / 6°带）
- 中央经线自动计算或手动配置
- 坐标系统自动转换

**定位功能**
- 实时定位显示
- 定位精度显示
- 基于位置自动计算中央经线

**配置系统**
- app.config.json 配置文件
- 作者信息和链接独立控制显示
- 可自定义应用名称、描述
- 可配置坐标系统、地图、文件、性能、检测等参数

#### 💄 界面优化
- 响应式设计，支持移动端和桌面端
- 文件名过长自动截断显示
- 点位名称支持多行显示
- 优化各组件布局和交互

#### ⚡ 性能优化
- 大文件虚拟滚动
- 点位聚合优化
- IndexedDB 数据存储
